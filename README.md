# Data Analytics Dashboard Sunrise Hotels

This is our group project for building a data dashboard for Sunrise Hotels.  
The idea is to bring data from different hotel systems into one place so managers can see what is happening in real time.

---

## 1. Project Summary

- Company: Sunrise Hotels (SH)  
- Goal: Make a dashboard that shows real-time business information  
- Why: Right now data is separate in different systems, so reporting is slow and manual  
- What the dashboard will help with:
  - Faster decisions
  - Less manual reporting
  - Better view of branches and departments
  - Improved guest experience
